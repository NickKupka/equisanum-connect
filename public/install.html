🍎 iPhone / iPad Installation über Safari
  1. Öffne den Link oben in Safari
  2. Tippe auf das ⬆️ Teilen-Symbol (unten in der Mitte)
  3. Scrolle und wähle „Zum Home-Bildschirm"
  4. Tippe auf „Hinzufügen" – fertig! ✨
🤖 Android Installation über Chrome
  1. Öffne den Link oben in Chrome
  2. Tippe auf die ⋮ drei Punkte (oben rechts)
  3. Wähle „App installieren" oder „Zum Startbildschirm"
  4. Bestätige mit „Installieren" – fertig! ✨
